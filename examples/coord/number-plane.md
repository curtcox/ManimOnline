# `coord/number-plane`

Show a full grid plane coordinate system.

Docs: [NumberPlane](https://docs.manim.community/en/stable/reference/manim.mobject.graphing.coordinate_systems.NumberPlane.html)

```py
from manim import *

class NumberPlaneExample(Scene):
    def construct(self):
        plane = NumberPlane()
        self.add(plane)
```
