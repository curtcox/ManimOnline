# `coord/complex-plane`

Plot complex numbers on a specialized plane.

Docs: [ComplexPlane](https://docs.manim.community/en/stable/reference/manim.mobject.graphing.coordinate_systems.ComplexPlane.html)

```py
from manim import *

class ComplexPlaneExample(Scene):
    def construct(self):
        plane = ComplexPlane()
        self.add(plane)
```
