# `cli/quality-flags`

Use `-ql/-qm/-qh` quality presets.

Docs: [Output & config tutorial](https://docs.manim.community/en/stable/tutorials/output_and_config.html)

```bash
manim -ql example.py BasicScene  # low quality
manim -qm example.py BasicScene  # medium quality
manim -qh example.py BasicScene  # high quality
```
