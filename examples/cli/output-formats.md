# `cli/output-formats`

Produce MP4/WebM/GIF/PNG outputs.

Docs: [Output & config tutorial](https://docs.manim.community/en/stable/tutorials/output_and_config.html)

```bash
manim -pql --format=mp4 example.py BasicScene
manim -pql --format=webm example.py BasicScene
manim -pql --format=gif example.py BasicScene
manim -pql --format=png example.py BasicScene
```
